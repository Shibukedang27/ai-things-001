# OfferPilot AI Brand Guidelines

OfferPilot AI is positioned as a premium SaaS product for interview intelligence, candidate readiness, and talent preparation. The brand should feel sharp, credible, data-driven, and calm under pressure.

## Brand Foundation

| Element | Direction |
| --- | --- |
| Product Name | OfferPilot AI |
| Brand Promise | Turn interview preparation into measurable offer readiness. |
| Voice | Clear, confident, concise, professional, optimistic. |
| Audience | Candidates, bootcamps, coaching companies, recruiting teams, universities, and enterprise talent teams. |

## Color Palette

| Token | Color | Hex | Usage |
| --- | --- | --- | --- |
| Primary Color | Orbit Navy | `#0B1220` | Headers, navigation, hero surfaces, premium brand blocks |
| Secondary Color | Signal Teal | `#14B8A6` | Primary actions, active states, charts, progress indicators |
| Accent Color | Offer Gold | `#F59E0B` | Highlights, conversion moments, pricing emphasis |
| Success Accent | Pilot Lime | `#A3E635` | Positive status, achievements, growth signals |
| Text | Ink | `#101828` | Primary body and heading text in light mode |
| Muted Text | Slate | `#667085` | Captions, metadata, secondary labels |
| Light Surface | Cloud | `#F8FAFC` | Page backgrounds and neutral bands |
| Dark Surface | Deep Console | `#111827` | Dark mode cards and elevated panels |

## Typography

- Primary font stack: `Inter, ui-sans-serif, system-ui, -apple-system, BlinkMacSystemFont, "Segoe UI", sans-serif`.
- Headings use high weight, tight hierarchy, and no negative letter spacing.
- Body copy should stay readable at 15px to 18px with generous line height.
- Product metrics and dashboard values can use heavier weights for quick scanning.

## Button Style

- Primary buttons use Signal Teal backgrounds, white text, 8px radius, and subtle lift on hover.
- Secondary buttons use transparent or surface backgrounds with a visible border.
- Icon buttons use lucide icons, 42px minimum target size, 8px radius, and hover color changes.
- Buttons should use verbs: `Start free`, `View dashboard`, `Book demo`, `Run analysis`.

## Cards

- Cards use 8px radius, 1px borders, and restrained shadows.
- Use cards for repeated items, pricing tiers, dashboard panels, testimonials, and contained tools.
- Avoid cards inside cards.
- Dense operational screens should favor scanability over oversized marketing layouts.

## Icons

- Use lucide-react for application and marketing icons.
- Use consistent stroke width between 1.8 and 2.2.
- Icons should clarify workflows: readiness, scoring, roadmap, coding, resume, analytics.
- Decorative icons should be minimal and tied to product meaning.

## Spacing

- Base spacing unit: 4px.
- Common spacing: 8px, 12px, 16px, 24px, 32px, 48px, 72px.
- Dashboard panels use tighter spacing for repeat workflows.
- Landing sections use larger vertical rhythm while keeping the next section visible.

## Theme

OfferPilot AI supports both light and dark modes. The interface should feel native in both, not like a simple inversion.

## Light Mode

- Background: `#F8FAFC`.
- Surfaces: `#FFFFFF`.
- Text: `#101828`.
- Borders: `#D0D5DD`.
- Primary action: `#14B8A6`.
- Accent moments: `#F59E0B`.

## Dark Mode

- Background: `#060A12`.
- Surfaces: `#111827`.
- Raised surfaces: `#172033`.
- Text: `#F8FAFC`.
- Muted text: `#A7B2C2`.
- Primary action: `#2DD4BF`.
- Accent moments: `#FBBF24`.
