# Tests

The test suite covers the backend foundations and intelligence engines built so far.

Current coverage includes:

- Backend import and OpenAPI sanity checks
- Document ingestion APIs
- Knowledge Engine pipeline behavior
- Knowledge DNA engine and APIs
- Multi-agent orchestration and APIs
- Hybrid retrieval, reasoning, performance, and stress behavior
- Interactive knowledge graph engine, API lifecycle, validation, and performance
- AI Workspace and Second Brain engine behavior plus API lifecycle
- Adaptive Learning engine behavior, API lifecycle, and performance

Run from the project root:

```bash
pytest tests -q
```
